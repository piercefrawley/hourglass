# boilerplate
React, Redux, Koa Boilerplate

## Getting Started

#### Clone
`git clone https://github.com/piercefrawley/boilerplate.git`
`cd boilerplate`

#### Install
`npm install`

#### Build
`npm run build`

#### Koa Webserver (localhost:3000)
`npm run webserver`

#### Boot App (localhost:8080)
`npm run dev`
